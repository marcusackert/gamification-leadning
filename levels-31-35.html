Level 31: Train the Knight

Define a class Knight.

Classes are blueprints for heroes. Use class followed by the name to begin training.

Level 32: Arm the Knight

Add __init__(self, name, strength) to store attributes.

The constructor equips the knight. Use __init__ to assign values to self.

Level 33: Battle Cry

Add method attack(self) that returns a phrase.

Methods are actions your hero can perform. Define one inside the class to make the knight shout.

Level 34: Lineage

Create Mage inheriting from Character.

Inheritance passes traits down. Use parentheses after the class name to show its parent.

Level 35: Many Forms

Override speak() in Mage to return a magical phrase.

Polymorphism lets different heroes speak differently. Redefine the method in the child class.
